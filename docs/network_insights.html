üï∏Ô∏è What is a Network?

A network shows relationships between entities. In our case:

  • Nodes (circles): Authors and research methods
  • Edges (lines): Connections showing "author uses method"
  • Size: Larger nodes = more connections (more important)

üîó Network Density

How connected the network is overall:

  • High density: Many connections between nodes
  • Low density: Few connections, isolated groups
  • RAND's network: High density = good collaboration
